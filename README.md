## Dataset Preparation
Download HEVC standard video sequences and encode them.
Arrange them in this way:
```
|YUV
|-- ClassA
|   |-- PeopleOnStreet_2560x1600_30.yuv
|   |-- Traffic_2560x1600_30.yuv
|-- ClassB
|   |-- ...
|-- ...
|pred
|-- ClassA
|   |-- qp22_AI
|   |   |-- PeopleOnStreet_2560x1600_30.yuv
|   |   |-- Traffic_2560x1600_30.yuv
|   |-- qp22_LD
|   |   |-- ...
|   |-- qp22_LDP
|   |   |-- ...
|   |-- qp22_RA
|   |   |-- ...
|   |-- qp27_AI
|   |   |-- ...
|   |-- ...
|   |-- qp37_RA
|   |   |-- ...
|rec
|-- ClassA
|   |-- qp22_AI
|   |-- | -- ...
|   |-- ...
|-- ...
|-- ClassB
|   |-- ...
```
